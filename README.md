# Resonance（無料版）

DLsite・FANZA・DiGiket 等で購入した音声作品をローカルで管理・再生するための Windows デスクトップアプリです。

[![Latest Release](https://img.shields.io/github/v/release/mrindigo24-design/resonance-releases?label=Latest&color=brightgreen)](https://github.com/mrindigo24-design/resonance-releases/releases/latest)
[![Release Date](https://img.shields.io/github/release-date/mrindigo24-design/resonance-releases?label=Released&color=orange)](https://github.com/mrindigo24-design/resonance-releases/releases/latest)
[![Downloads](https://img.shields.io/github/downloads/mrindigo24-design/resonance-releases/total?label=Downloads&color=blue)](https://github.com/mrindigo24-design/resonance-releases/releases)

> 📦 **これは無料版です** — 登録できる作品数が **200 作品まで** に制限されています。  
> 制限なしで利用したい方は、有料版（Resonance Pro）が販売予定です（販売 URL は今後追記）。  
> 無料版で登録したデータは、有料版インストール後もそのまま引き継がれます。

## インストール方法

1. ダウンロードした `resonance-free-X.Y.Z-setup.exe` をダブルクリックして実行
2. インストーラーの指示に従ってインストール（インストール先はデフォルトのままで OK）
3. インストール完了後、デスクトップまたはスタートメニューから「Resonance」を起動

デフォルトのインストール先は `%LOCALAPPDATA%\Programs\Resonance\` で、**管理者権限は不要**です。
データはインストールフォルダではなく、Windows の `%APPDATA%\Resonance` 配下に保存されます。

### アンインストール
Windows の「設定」→「アプリ」→「Resonance」から、または「コントロールパネル」→「プログラムのアンインストール」から削除できます。

## 使い方の基本

1. アプリを起動
2. サイドバーの「**+ Import Folder**」または「**Scan Directory**」で作品フォルダを取り込み
3. 取り込まれた作品をクリックして再生
4. タグ・CV・サークルでフィルタリング、スマートフォルダで分類

詳細な使い方は、アプリ本体に同梱されている `MANUAL.html` を参照してください。

## 主な機能

### ライブラリ管理
- フォルダスキャンによる作品の自動取り込み
- **多サイト対応** — DLsite / FANZA / DiGiket はプリセットで判別、独自サイトもサイトテンプレートで登録可能
- 作品ページの URL を貼り付けるだけでサイトと作品コードを自動判別
- カバー画像の自動検出
- メタデータ（タイトル・サークル・CV・タグ・評価など）の編集
- 重複登録チェック
- バックアップ・リストア（**有料版のみ**）

### 検索・フィルタ
- タグ・CV・サークル・シリーズによる絞り込み
- 評価・お気に入り・再生状態による絞り込み
- スマートフォルダ（複雑な条件を組み合わせた仮想フォルダ）
- 姉妹サークル（別名義の関連サークルをまとめて表示）
- サイドバーのフィルタセクションを個別に表示/非表示切替可能

### 音声プレイヤー
- 高機能な再生コントロール（シーク・スキップ・速度変更）
- AB ループ（区間リピート）
- スリープタイマー
- LRC 歌詞ファイルの同期表示
- オーバーブースト・音割れ防止リミッター
- 一時停止時の自動リソース解放

### プレイリスト・キュー
- 作品内プレイリスト
- グローバルプレイリスト
- 再生キュー（作品をまたいで連続再生）

### 高度な再生機能
- **分岐再生**: トラック終了時に選択肢を表示してマルチエンディング体験
- **シナリオ分岐**: 作品ごとの変数（難易度・ルート等）で分岐の出現条件を制御
- **クイックジャンプ**: 数字キーで特定トラックへ即座にジャンプ
- **次トラック設定**: 特定トラックの後に固定のトラックへジャンプ
- **トラック整理**: カスタム並び順・仮想フォルダ・表示名カスタマイズ

### ドキュメント閲覧
- 作品フォルダ内の `.txt`（台本・キャラ設定）と `.pdf`（小説・ブックレット等）を「ドキュメント」タブから一覧・閲覧
- テキストは UTF-8 / Shift-JIS / EUC-JP など各種エンコーディングを自動判定
- 文字サイズ・行間・色をカスタマイズ可能（セピア / ダーク等のプリセット付き）
- PDF は Chromium 内蔵ビューアでズーム・検索・ページ送り対応
- ドキュメントを見ながら詳細パネルでメタデータ編集・再生操作も可能

### タグ・サークル管理
- タググループ・色分け・リネーム・一括削除
- CV エイリアス（別名統合）・お気に入り
- サークル・シリーズの一覧管理
- **サークル / シリーズの複数サイト URL** 登録（サイトごとに URL を保持）
- 姉妹サークルの登録（双方向・推移的展開）

### 配布対応
- `.rsce` ファイル（分岐・シナリオ設定）の自動インポート
- `.rsdate` ファイル（作品メタデータ）の自動インポート
- 複数サイトの URL は**スマートマージ**（他サイトの URL を壊さず統合）
- 制作者が分岐設定済みの作品をそのまま体験可能

### メタデータの自動エクスポート
- 各作品フォルダに `resonance.rsdate` / `resonance.rsce` を自動生成する機能（既定 OFF）
- PC 移行・フォルダ移動後もフォルダを再インポートするだけでメタデータが**自動復元**
- 全作品の一括書き出しボタンも用意

### 自動アップデート
- 新しいバージョンが公開されると、起動時に自動で通知
- 環境設定 → アップデート タブから手動確認・適用が可能
- 起動時の自動チェックは ON/OFF 切替可能
- 差分アップデート対応で、更新時のダウンロードサイズを大幅削減
- アップデート適用時はインストーラー画面を表示せず、バックグラウンドで静かに更新

## アップデート方法

新しいバージョンが公開されると、アプリ起動時にサイドバーの「ツール」パネル下部のバージョン表記にバッジが表示されます。
環境設定 → 「アップデート」タブを開いて「ダウンロード」を押すと自動的に更新されます。

## 既知の制限

- **無料版は 200 作品まで** しか登録できません（201 作品目以降は取り込みが阻止されます）
  - 既に 200 作品を超えるデータベースを開いた場合、作品メタデータの編集がブロックされます。200 作品以下まで削除すると編集が再開されます
  - 制限なしで利用したい場合は、Resonance Pro（有料版）をご検討ください
- macOS / Linux 版は提供していません
- DRM 付き音声ファイルの再生には対応していません
- スキャン対象は指定フォルダの **直下のサブフォルダ** が 1 作品として認識される構成のみ
  （入れ子構造には非対応）
